№1

Расскажите, какого размера будут заголовки после выполнения следующего кода:

<h2>text</h2>
h2 {
	font-size: 20px;
}
h2 {
	font-size: 30px;
}

Ответ: 30px, т.к вторая запись переопределит первую.

№2
Расскажите, какие свойства применятся к заголовкам и какие будут значения этих свойств:

<h2>text</h2>
h2 {
	font-size: 20px;
	color: blue;
}
h2 {
	font-style: italic;
	color: red;
}

Ответ: заголовок будет отображаться красным курсивом с размером 20px.
