В данном коде есть следующие родительские связи:
  • <a> - родительским элементом для тега <b> является <a>.
  • <li> - родительским элементом для каждого <a> (включая <b>) является <li>.
  • <ul> - родительским элементом для всех <li> является <ul>.

В данном коде есть следующие родительские связи:
  • <td> - родительским элементом для каждой <td> является <tr>.
  • <tr> - родительским элементом для каждой <tr> является <table>.
  • <table> - является корневым элементом в данной структуре.

-✧༺⋞ 〈 "№3" 〉 ⋟༻✧-

У абзаца (<p>) есть следующие потомки:
  1. Текст "текст абзаца".
  2. <i> элемент, внутри которого находится жирный текст "жирный курсив".
  3. <a> элемент, внутри которого есть следующие потомки:
    • Текст "ссылка ".
    • <i> элемент с текстом "курсив".
Непосредственными потомками абзаца (<p>) являются только три элемента: текст "текст абзаца", <i> и <a>.

В данном коде у вас есть следующие родительские связи:
  • <a> - родительским элементом для <i> и <b> является <a>.
  • <i> - родительским элементом для <b> является <i>.
Непосредственным родителем для <b> является <i>.
